## Getting Started

If you would like to clone and run the project locally, follow the steps below.

### Installation

1. Clone the repo

```sh
git clone git@github.com:thnlsn/portfolio.git
```

2. Install NPM packages

```sh
npm install
```

3. Run the app!

```JS
npm start
```

4. Check it out at: http://localhost:3000/portfolio
